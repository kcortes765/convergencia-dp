# Estudio numérico de inicio de movimiento

Sitio estático público con la síntesis metodológica de la tesis:

- convergencia de resolución SPH;
- hidráulica local antes del bloque;
- modelo GP de frontera e incertidumbre;
- resultados SPH observados;
- incertidumbre de resolución;
- estado metodológico antes de Monte Carlo.

El sitio publicado contiene solo páginas HTML y figuras visibles en esas páginas.
