Qué predice el modelo

El modelo GP interpola el margen g = 5 - Dmax. Si g > 0, el desplazamiento queda bajo el umbral de movimiento; si g < 0, el caso supera el umbral. La incertidumbre del GP sirve para interpolar la frontera y luego alimentar una lectura probabilística.

Lectura de validación: el porcentaje de clase correcta mide consistencia predictiva caso a caso. Es suficientemente útil para interpolar y propagar incertidumbre, pero los errores cerca del umbral se deben leer junto con MAE, RMSE y la banda de resolución.
